# _wrk

This directory contains code from older repositories being evaluated for integration into **minimo**.

Each item may be:
- **integrated** into minimo as-is or after refactoring
- **rewritten** to fit minimo's conventions
- **discarded** if deemed outdated or no longer useful

Nothing here should be considered stable or production-ready.
